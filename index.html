Stardew Community Center Checklist

Welcome! This website checks your Stardew Valley save file and gives you a checklist for completing the Community Center bundles in your first year. This checklist will only work with saves using default (not remixed) bundles and is recommended to be used in your first year.

File Upload

Please select your Stardew Valley save file:

Please upload your save file to recieve your checklist. The save file is titled with your farmer's name (or farm name) followed by your ID number. Please note that your save file is not titled "SaveGameInfo". Checklist may not make sense if you upload a save that uses remixed bundles:

Format: [name]_[#########] (e.g. JonSnow_123456789)
Default save locations:
  • Windows: %appdata%\StardewValley\Saves
  • Linux/MacOS: ~/.config/StardewValley/Saves
For more information on finding your save file go to Saves.
